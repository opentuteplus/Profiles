Profiles helps site builders and WordPress developers add Profiles features to their websites, with user profile fields and members directory.

== Description ==

Are you looking for modern, robust, and sophisticated Profiles Builder? Profiles is a suite of components that are for building user profiles.

Profiles is focused on ease of integration, ease of use, and extensibility. It is deliberately powerful yet unbelievably simple Profiles software, built by OpenTute+.

Enable registered members to create profiles.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'Profiles'
3. Activate Profiles from your Plugins page.
= From WordPress.org =

1. Download Profiles.
2. Upload the â€˜profiles' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate Profiles from your Plugins page.
